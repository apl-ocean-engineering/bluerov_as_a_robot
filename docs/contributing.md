# Contributing

This document was created as a way to share best practices and provide a jump-start for new BlueOS robotics projects.  Obviously we don't know everything and BlueOS will continue to evolve.  We welcome contributions of new content and tip, and notes where things don't work as expected.

To contribute:

* Fork [this repo]() from Github
* Make changes in a [feature branch]()
* Be sure to use [`pre-commit`](#pre-commits)!
* Submit a pull request.

## Pre-commit

We use [pre-commit]() to regularize formatting.  Please install and use pre-commit!  Changes which do not pass pre-commit in CI will be sent back.
